Ruby Scorekeeper

Description:

I built this app with Ruby. The purpose is for keeping track of players' scores. I was inspired to do this while playing a dice game called "Farkle". This particular game has a set winning score, though I also made the app adaptable for other games that don't have a set winning score. So you can use it for other dice games, like Yahtzee, or various card games, like Hand & Foot. It's currently a command-line program, though my next step is to utilize Sinatra or Rails to create a live scoreboard page that will be much more visually appealing.
